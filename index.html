JFTL 2015 : fiche technique

Hiptest : https://hiptest.net

Jira : https://hiptest.atlassian.net/

Cloud9 : www.c9.io

Cloner les sources : https://github.com/severinedarlot/jftl2015

Mastering: http://littlebigmusic.eu/calculator/

Configuration environnement de développement :

npm install -g ember-cli@v0.2.0

npm install -g bower

npm install -g phantomjs

npm install

bower install

ember server

Voir l’application : https://jftl2015-login.c9.io/ ou bouton ‘preview’

Générer les tests :

gem install hiptest-publisher

Mettre à jour le token dans le fichier de configuration ./hiptest-publisher.config

Génération des tests :

hiptest-publisher -c ./hiptest-publisher.config

hiptest-publisher -c ./hiptest-publisher.config --tests-only

hiptest-publisher -c ./hiptest-publisher.config --actionwords-only

Lancer les tests

depuis la console : ember test

depuis un navigateur : https://jftl2015-login.c9.io/tests

Automatisation

dans le fichier tests/integration/features/actionwords.js

Intégration continue :

hiptest-publisher -c ./hiptest-publisher.config --tests-only

ember test > results.tap

hiptest-publisher -c ./hiptest-ci.config -p results.tap
